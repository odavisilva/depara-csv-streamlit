# Depara CSV com Streamlit

Aplicacao Python para comparar dois arquivos CSV e exibir divergencias de forma clara.

## Requisitos

- Python 3.10+

## Instalacao

```bash
pip install -r requirements.txt
```

## Execucao

```bash
streamlit run app/main.py
```

## Como usar

1. Envie o CSV A e o CSV B.
2. A aplicacao valida se ambos sao `.csv`.
3. Veja o resumo de comparacao e a tabela de diferencas.
4. Baixe o relatorio em CSV.
